# Editing and Manipulating Images with Midjourney via Playwright

Use Midjourney's editor and image reference features to refine, upscale, extend, or transform existing images -- including images generated by other tools like fal, DALL-E, or Stable Diffusion.

## Prerequisites

- Playwright MCP plugin must be installed and available
- User must have an active Midjourney subscription
- User must be logged in to midjourney.com

## Use Cases

### Upscaling / Super-Resolution
For images that need higher resolution:

1. Upload the image to Midjourney as an image prompt
2. Use a minimal text prompt describing what's in the image
3. Set `--iw 2` or `--iw 3` (high image weight) to stay faithful to the original
4. Generate, then use **Upscale > Subtle** for clean 2x resolution boost

### Background Removal / Replacement
1. Upload the image to the Midjourney editor
2. Use the **Erase brush** to paint over the background
3. Write a new prompt describing the desired background (or "transparent background, isolated subject")
4. Submit edit

### Outpainting / Extending Canvas
To make an image wider, taller, or extend in any direction:

1. Open the image in Midjourney's editor at `/editor/new`
2. Use the **resize handles** to drag edges outward in the desired direction
3. Select a new aspect ratio if needed (e.g., 16:9 from a square)
4. Write a prompt that describes what should fill the extended area
5. Submit edit -- Midjourney will seamlessly generate the extended regions

### Inpainting / Object Removal
To remove or replace specific elements:

1. Open image in the editor
2. Use **Erase brush** or **Smart Select** to select the area to change
3. Write a prompt describing what should replace it (or leave blank to remove)
4. Submit edit

### Retexturing / Style Transfer
To change the visual style while keeping composition:

1. Open image in the editor
2. Switch to the **Retexture** tab
3. Write a prompt describing the new style (e.g., "watercolor painting", "cyberpunk neon")
4. Submit -- Midjourney retextures while preserving structure

### Using Images as References
To generate new images influenced by an existing one:

**As image prompt** (composition + content influence):
```
https://uploaded-image-url.png a majestic landscape --iw 1.5
```

**As style reference** (style only):
```
a portrait photograph --sref https://uploaded-image-url.png --sw 200
```

**As character/omni reference** (character consistency):
```
running through a forest --oref https://uploaded-image-url.png --ow 200
```

## Uploading Images to Midjourney

### Via the Prompt Bar
1. Click **Add Images** button (left side of prompt bar)
2. Upload the image file or paste a URL
3. The image appears as a thumbnail in the prompt area
4. Add text prompt and parameters after the image

### Via the Editor
1. Navigate to `https://www.midjourney.com/editor/new`
2. Upload an image to start editing
3. Use editor tools (erase, resize, retexture) to modify

### Getting a URL for Local Images
If you have a local image file, you can upload it to Midjourney's prompt bar directly using the "Add Images" button. For URL-based workflows, the image needs to be accessible via URL.

## Editor Navigation

| URL | Purpose |
|-----|---------|
| /editor/new | Start fresh with uploaded image |
| /edit/{job-id}?index=N | Edit an existing Midjourney generation |

## Editor Tools

| Tool | Purpose |
|------|---------|
| Erase brush | Paint areas to regenerate (inpainting) |
| Restore brush | Undo erased areas |
| Smart Select | AI-powered region selection |
| Resize handles | Drag edges to extend canvas (outpainting) |
| Retexture tab | Apply new styles preserving structure |

## Aspect Ratio Presets (Editor)
1:1, 3:4, 2:3, 9:16, 1:2, 4:3, 3:2, 16:9, 2:1

## Tips for Editing External Images
- **High image weight** (`--iw 2` or `--iw 3`) keeps the output faithful to your source
- **--raw** mode helps preserve photorealistic qualities from the original
- Use **Upscale > Subtle** (not Creative) when you want to preserve the original look
- For images from fal/SD/DALL-E that need cleanup, the editor's inpainting is great for fixing hands, faces, or artifacts
- Retexture is powerful for converting between styles while keeping layout intact
